☁ Sincronização com JSONbin

Como configurar (uma vez por dispositivo):
1. Crie conta grátis em jsonbin.io (10k requests/mês)
2. Crie um Bin: clique em "New Bin" → cole {"data":[]} → Save
3. Copie o Bin ID (no topo da página do bin, formato 67abc123...)
4. Em Settings → API Keys, crie um Access Key com permissão de leitura+escrita
5. Cole abaixo e salve. Faça o mesmo nos outros dispositivos com o mesmo Bin ID e Access Key.
